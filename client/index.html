Valuation Model Overview

Technical summary of our machine learning pipeline.

Our backend serves a Linear Regression model trained using Scikit-Learn. The algorithm resolves independent variables: total square footage, bathroom count, bedrooms (BHK), and location coefficients generated via one-hot encoding to compute predicted prices.

The model is trained on the Kaggle Bengaluru House Price dataset containing over 13,300 structural records. We applied statistical cleanup processes, including removal of outliers based on Price per Square Foot and BHK-to-bathroom ratios, ensuring estimates stay realistic.

If the Flask API server is offline, the client seamlessly shifts to the local simulator mode. It executes a local deterministic algorithm using location-based weight hashes to estimate prices, ensuring you can evaluate and inspect the user interface without having backend dependencies active.
